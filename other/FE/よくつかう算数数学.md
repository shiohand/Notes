## 指数

|         - | -   | -         |
| --------: | --- | --------- |
| a^m * a^n | =   | a^(m + n) |
| a^m / a^n | =   | a^(m - n) |
|   (a^m)^n | =   | a^(mn)    |
|   a^(1/x) | =   | a^(xn/n)  |
|       a^0 | =   | 1         |
|       a^1 | =   | a         |
|      a^-n | =   | 1/a^n     |

* 分数の指数 まあ過去問にある分だけ覚えたらいいと思うけど

|       - | -                 | -          |
| ------: | ----------------- | ---------- |
| a^(1/2) | (a^(1/2))^2 = a   | aの平方根  |
| a^(1/x) | (a^(1/x))^x = a   | aのx乗根   |
| a^(y/x) | (a^(y/x))^x = a^y | a^yのx乗根 |

## 計算式

### 実行アクセス時間
キャッシュメモリを使った実行アクセス時間
```
ヒット率 * キャッシュメモリのアクセス時間 + (1 - ヒット率) * メインメモリのアクセス時間
```

### MIPS
命令回数(百万) / 1秒
