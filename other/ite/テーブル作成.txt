テーブル作成

/* ALTER TABLE
---------------------------------------- */
ALTER TABLE データベース.テーブル
  ADD カラム名 データ型 制約,
  ADD PRIMARY KEY(カラム1, カラム2, ...),
  DROP カラム名,
  DROP PRIMARY KEY,
  CHANGE COLUMN カラム名 新しいカラム名 データ型 制約;

※ADD PRIMARY KEY(...カラム名), DROP PRIMARY KEY; で主キーの追加削除
※CHANGE COLUMNでは変更する制約や名前だけでなく全ての情報を指定すること
