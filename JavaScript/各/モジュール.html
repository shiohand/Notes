モジュールの利用 ・宣言不要で 'use strict'; 扱い ・インポートされたモジュール同士は別スコープ 他のモジュールのプロパティなどは呼べない 自分のファイル中でimportすれば当然使える モジュール, 導入 https://ja.javascript.info/modules-intro export (変数、定数、関数などの宣言前につける) 最上位のオブジェクトのみ 関数内で宣言したオブジェクトには付けられない import ({...受け取るオブジェクト} from 'パス') パスは拡張子をつけないとかつけるとかわからん 他のimport export default ファイル内に要素が一つしかなかったら自動でそれにきまるので、 関数名とかクラス名とかもいらない インポートガワで名前をつける {}不要
